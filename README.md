Разместил проект на vercel по адрессу https://aiti-guru-test-three.vercel.app/

Проект написан на архитектуре FSD с применением следующих технологий:

### 1. Ядро приложения
react / react-dom: Фундамент для построения интерфейса.

react-router-dom: Управление маршрутизацией в приложении.

### 2. Управление состоянием и данными
tanstack/react-query: Для работы с серверным состоянием (запросы, обновление данных).

zustand: В качестве state менеджера.

### 3. Работа с формами и валидация
react-hook-form: Для работы с состоянием форм.

zod: Библиотека для схемы данных, используется для валидации форм.

### 4. Сетевые запросы и утилиты
axios: Клиент для выполнения HTTP-запросов.

js-cookie: Для работы с cookie

react-hot-toast: для уведомлений.

### 5. Стилизация и сборка
sass: Препроцессор CSS.

vite: Инструмент сборки.
